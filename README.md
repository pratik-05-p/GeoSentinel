# GeoSentinel

## AI-Powered Geological Landslide Hazard Assessment Platform

GeoSentinel is a web-based landslide hazard assessment platform that enables rapid preliminary risk evaluation using geological and environmental parameters. The system analyzes multiple factors influencing slope stability and generates hazard scores, risk categories, mitigation recommendations, and downloadable PDF reports.

---

## Problem Statement

Landslides are among the most destructive geological hazards, causing loss of life, infrastructure damage, and economic disruption worldwide.

Traditional landslide risk assessment often requires specialized software, expert analysis, and significant time. GeoSentinel addresses this challenge by providing an accessible platform for rapid preliminary landslide susceptibility assessment.

---

## Features

* Landslide Hazard Score Calculation
* Risk Categorization (Low, Moderate, High, Critical)
* Geological Factor Analysis
* Mitigation Recommendations
* PDF Report Generation
* Assessment History Tracking
* Responsive Dashboard Interface

---

## Input Parameters

GeoSentinel evaluates landslide susceptibility using:

* Slope Angle
* Rainfall Intensity
* Soil Type
* Vegetation Cover
* Fault Zone Proximity
* Land Use Type
* Drainage Condition

---

## How It Works

1. User enters geological and environmental parameters.
2. GeoSentinel computes a hazard score using a weighted assessment model.
3. The system classifies the location into a risk category.
4. Hazard factors are analyzed and explained.
5. Mitigation recommendations are generated.
6. A PDF assessment report can be downloaded.
